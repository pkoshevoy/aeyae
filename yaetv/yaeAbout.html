Acknowledgments

yaetv is a Digital Video Recorder compatible with HDHomeRun tuners. It relies heavily on the excellent FFmpeg APIs. Windows builds are possible thanks to Zeranoe. Video rendering is done with OpenGL, GLEW and Qt. Windows installer is built using WiX 3.5. Cross platform build configuration is put together using CMake. JSON support is thanks to jsoncpp.
