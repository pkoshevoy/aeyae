Acknowledgments

Apprentice Video is an open source video player. It relies heavily on the excellent FFmpeg APIs. Windows builds are possible thanks to Zeranoe. Audio rendering is implemented via PortAudio. Video rendering is done with OpenGL and Qt4. ASS/SSA subtitle rendering is implemented via libass. Apple Remote Control support is thanks to Martin Kahr. Windows installer is built using WiX 3.5. Cross platform build configuration is put together using CMake.

Keyboard Shortcuts

NOTE: for OS X, use Command key in place of Ctrl key in the shortcuts below.

Space Bar
Play, or pause. If automatic bookmark creation has not been disabled, then a bookmark for the current playlist group will be created or updated whenever playback is paused.

Ctrl + P
Show or hide the playlist.

Alt + Left Arrow
Go back to previous playlist item.

Alt + Right Arrow
Skip to next playlist item.

Ctrl + N
Skip to next chapter, or next playlist item.

Ctrl + A
Select all items in the playlist.

Del
Remove currently selected item(s) from the playlist.

Ctrl + F
Switch to letterbox full screen mode with black pillars or bars padding the scaled image to fit the screen.

Ctrl + G
Switch to full screen mode with image scaled to fill the screen and the excess image cropped off.

Esc
Exit (any) full screen mode.

Ctrl + R
In window mode this will shrink the window to wrap the video frame to eliminate any slack.

Ctrl + 0
In window mode this will resize the window to half the size of the video frame.

Ctrl + 1
In window mode this will resize the window to the size of the video frame.

Ctrl + 2
In window mode this will resize the window to double the size of the video frame.

Alt + 4
Crop video to 4:3 aspect ratio.

Alt + 6
Crop video to 16:9 aspect ratio.

Alt + 8
Crop video to 1.85 aspect ratio.

Alt + 2
Crop video to 2.40 aspect ratio.

Alt + C
Automatically crop video to eliminate black borders, pillars and bars. This is not a continuous operation, so if the borders change during playback of the current playlist item they will not be automatically detected and eliminated. In this case repeat auto crop manually. However, the operation will be repeated automatically when playback switches to another playlist item.

Alt + X
Reset video crop settings to initial state (no cropping).

Alt + Shift + 4
Force video to 4:3 aspect ratio.

Alt + Shift + 6
Force video to 16:9 aspect ratio.

Alt + Shift + X
Set video to default (automatic) aspect ratio.

Ctrl + T
Show or hide the timeline.

I
Set timeline loop in-point.

O
Set timeline loop out-point.

N
This is a crude frame step feature. When playback is paused this will advance the current playhead position by one (average) frame duration.

.
>
Skip forward 7 seconds.

,
<
Go back 3 seconds.

Ctrl + Q
Quit the player.

Playlist Controls

Type in keywords in the playlist search box to filter down the playlist. This will also filter the bookmarks menu.

Move mouse towards the right edge of a playlist item where an [x] button will appear. Click it to remove the item (or item group) from the playlist.

Use Tab key to switch keyboard focus between playlist items and playlist search box.

Use Arrow keys to navigate the playlist items.

Other Controls

Right Click on the playlist or video canvas to open the context menu.

Double Click the video canvas to enter or exit the most recently used full screen mode (default is letterbox mode).

Double Click the timeline to seek.

Use the mouse Scroll Wheel to seek.

Click and Drag the timeline playhead to seek.

Click and Drag the timeline in/out triangular handles to set the timeline loop in/out points.

Click the playhead position display on the left side of the timeline, then type in timeline position in HH:MM:SS:FF format and press enter to skip to that position. Here FF stands for frame number, however frame-accurate seeking is not guaranteed.

Apple Remote Control, for OS X only

Click the Right button to skip forward 7 seconds.

Click the Left button to go back 3 seconds.

Click the Play/Pause button to play or pause.

Click the Menu button to show or hide the timeline.

Click-and-hold the Menu button for about a second to enable or disable automatic cropping of black borders, pillars and bars.
